# Abhiram Constructions

A premium, luxury construction company specializing in residential and commercial projects in **Hyderabad’s East Corridor (Uppal & Ghatkesar)**.

## 🏗️ Project Overview
This is a modern, performance-optimized website built with:
- **Core**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Tailwind CSS & Vanilla CSS
- **Animations**: Framer Motion
- **Knowledge**: Custom-built Blog/Insights hub with SSG optimization
- **Interactivity**: Dynamic Lead Generation, Interactive Cost Estimator, and WhatsApp integration

## 📂 Key Features
- **Masterpieces Gallery**: Interactive showcase of completed and ongoing projects.
- **Knowledge Hub**: Static-site-generated (SSG) articles for SEO and performance.
- **Lead Generation Hub**: A world-class form system designed for high conversion.
- **Process Storyteller**: An animated journey of the construction cycle.

## 🚀 Getting Started
```bash
npm install
npm run dev
```

## 🛠️ Deployment
This project is configured for **Vercel** with the **`main`** branch serving as the production environment.
- **Production URL**: [abhiramconstructions.vercel.app](https://abhiramconstructions.vercel.app)
